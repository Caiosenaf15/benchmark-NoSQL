Resumo

Esta página disponibiliza os artefatos utilizados no artigo "Benchmark do Impacto de Mecanismos Criptográficos em Bancos de Dados NoSQL".

O benchmark avalia o impacto de algoritmos criptográficos em três bancos de dados NoSQL:

  • MongoDB
  • Redis
  • Apache Cassandra

Foram avaliados os algoritmos:

  • AES-256
  • SHA-256
  • HMAC-SHA256
  • bcrypt
  • Argon2

Reprodução dos Experimentos

Os experimentos podem ser reproduzidos em sistemas Linux ou Windows utilizando os scripts disponibilizados no repositório.

1. Clonar o repositório 
git clone https://github.com/Caiosenaf15/benchmark-NoSQL.git
cd benchmark-NoSQL
2. Instalar dependências 
pip install -r requirements.txt
3. Executar os benchmarks 
bash scripts/run_all.sh
